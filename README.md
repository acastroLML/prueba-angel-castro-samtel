### Aplicacion web
Creacion Proyecto Vue.js


    $ npm install -g @vue/cli
    $ npm install -g npm@9.7.1
    $ vue create app-test-angel
    $ cd app-test-angel/
    $ npm run serve

    App running at:
    - Local:   http://localhost:8080/ 

    Ejecutamos el modo producción de Vue
    $ npm run build
